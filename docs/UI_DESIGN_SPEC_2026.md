# AI Quant Pro 2026 - UI/UX 设计规范与高保真原型方案

**版本**: 1.0  
**日期**: 2026-01-21  
**状态**: 待评审  
**设计目标**: 打造沉浸式、未来感的“量化资产管理操作系统 (Quant OS)”。

---

## 1. 设计理念 (Design Philosophy)

*   **沉浸 (Immersive)**: 全黑背景 + 霓虹光效，营造专业交易室氛围，降低长期盯盘的视觉疲劳。
*   **智能 (Intelligent)**: AI 助手不再是角落的图标，而是占据核心交互区的“副驾驶”。
*   **极简 (Minimalist)**: 隐藏非必要控件，数据可视化优先，用图形代替表格。
*   **玻璃拟态 (Glassmorphism)**: 关键悬浮层采用磨砂玻璃效果，体现层级感与现代感。

---

## 2. 视觉规范 (Visual Identity)

### 2.1 色彩系统 (Color Palette)

| 用途 | 色值 | 说明 |
| :--- | :--- | :--- |
| **背景色 (Deep Space)** | `#050505` | 极致深黑，略带冷色调，用于主背景 |
| **卡片底色 (Surface)** | `#141414` | 次级背景，用于模块承载 |
| **主色 (AI Neon)** | `#2979FF` | 科技蓝，用于主按钮、AI 交互点、选中态 |
| **强调色 (Cyber Purple)** | `#D500F9` | 赛博紫，用于“策略基因”、“突发信号” |
| **上涨 (Bull Green)** | `#00E676` | 高饱和荧光绿，用于盈利、买入信号 |
| **下跌 (Bear Red)** | `#FF1744` | 高饱和荧光红，用于亏损、卖出信号 |
| **文本 (Primary)** | `#FFFFFF` | 87% 透明度白色 |
| **文本 (Secondary)** | `#FFFFFF` | 60% 透明度白色 |

### 2.2 字体规范 (Typography)

*   **UI 字体**: `Inter`, `-apple-system`, `PingFang SC` (清晰、中性)
*   **数据/代码字体**: `JetBrains Mono`, `Roboto Mono` (等宽，便于数字对齐)
*   **字号**:
    *   H1 (页面标题): 24px, Weight 600
    *   H2 (模块标题): 18px, Weight 500
    *   Body (正文): 14px, Weight 400
    *   Data (核心指标): 32px, Monospace, Weight 700

### 2.3 布局原则 (Layout)

采用 **Bento Grid (便当盒布局)**，将屏幕划分为模块化的功能区块，支持拖拽重组。

*   **左侧**: 导航栏 (收缩图标)
*   **顶部**: 全局状态栏 (资金概览 + 市场情绪Ticker)
*   **中央**: 核心工作区 (图表/回放/代码)
*   **右侧**: AI 交互区 + 实时风控雷达 (Copilot Sidebar)

## 3. 核心界面高保真原型设计 (High-Fidelity Prototypes)

### 3.1 首页：AI 量化驾驶舱 (The Cockpit)
*   **设计意图**: 让用户一眼掌控全局，AI 随时待命。
*   **核心元素**:
    *   **AI 对话框 (Center Stage)**: 类似 ChatGPT 的输入框，“告诉我你想交易什么？”，支持语音输入。
    *   **策略卡片流 (Strategy Stream)**: 下方展示推荐的“策略基因组合”。
    *   **市场情绪云**: 背景动态粒子的颜色反映市场多空情绪。

### 3.2 策略分析页：时光穿梭机 (Time Machine)
*   **设计意图**: 游戏化的回测体验。
*   **交互**:
    *   底部设有播放控制条 (Play/Pause, 1x/2x/10x)。
    *   K线图上动态标注买卖点 (Buy/Sell Markers)。
    *   右侧“高光/至暗时刻”列表，点击直接跳转时间轴。

### 3.3 移动端适配 (Mobile Responsiveness)
*   **布局**: 单列流式布局。
*   **导航**: 底部 Tab Bar。
*   **交互**: 关键指标卡片支持左右滑动切换。

---

## 4. 交互流程 (Interaction Flow)

### 场景：从灵感到实盘
1.  **Input**: 用户在驾驶舱输入“想在这个震荡市里赚点钱，风险别太大”。
2.  **Process**: AI 唤起“策略医生”，分析当前市场波动率，推荐“布林带 + 网格”组合。
3.  **Display**: 屏幕展示策略回测概览，并高亮“最大回撤”风险点。
4.  **Action**: 用户点击“优化”，AI 自动调整网格间距。
5.  **Deploy**: 用户点击“模拟盘运行”，策略无缝部署。
