# User Interaction Flows (UX Flows)
**Version**: 1.0  
**Date**: 2026-01-21  
**Tool**: Mermaid JS

## 1. 基金智能筛选路径 (Fund Discovery Flow)

用户目标：从 10,000+ 基金中找到适合自己的“高收益、低回撤”基金，并进行深度诊断。

```mermaid
graph TD
    A[用户进入基金大厅] --> B{选择筛选方式}
    B -->|快速筛选| C[点击热门标签<br/>(e.g. 近一年涨幅Top10, 只有张坤)]
    B -->|高级筛选| D[设置参数<br/>(Sharpe>2, MaxDD<15%, 规模>20亿)]
    B -->|智能搜索| E[输入自然语言<br/>"找一只抗跌的科技基"]
    
    C --> F[基金列表结果页]
    D --> F
    E --> F
    
    F --> G{查看单只基金}
    G -->|悬停预览| H[Mini Chart & 核心指标]
    G -->|点击详情| I[基金深度透视页]
    
    I --> J[查看业绩归因]
    I --> K[查看经理能力雷达]
    I --> L[定投测算]
    
    L --> M[加入自选 / 模拟买入]
```

## 2. 股票深度研究路径 (Stock Deep Dive Flow)

用户目标：对某只异动股票进行全方位分析（技术面+基本面+资金面）。

```mermaid
graph TD
    A[行情看板/自选股] -->|点击个股| B[个股详情页 Dashboard]
    
    subgraph 左侧：行情与交易
    B --> C[查看分时/K线]
    C --> C1[切换周期 1m/5m/1D]
    C --> C2[叠加指标 MACD/BOLL]
    end
    
    subgraph 中间：深度数据
    B --> D[财务体检]
    D --> D1[杜邦分析图]
    D --> D2[PE/PB Band]
    B --> E[资金流向]
    E --> E1[北向资金持仓]
    E --> E2[主力大单监控]
    end
    
    subgraph 右侧：AI 辅助
    B --> F[AI 策略吹哨]
    F -->|触发信号| G[提示: RSRS指标发出买入信号]
    B --> H[智能研报摘要]
    end
    
    G --> I[一键回测该信号]
    I --> J[生成回测报告]
```

## 3. 资产配置驾驶舱路径 (Wealth Allocation Flow)

用户目标：查看全账户风险暴露，并进行再平衡。

```mermaid
graph TD
    A[资产总览] --> B[查看相关性矩阵]
    B --> C{发现风险点}
    C -->|相关性过高| D[警告: 权益类资产过于集中]
    
    A --> E[查看资产穿透图]
    E --> F[底层资产分布<br/>(股票/债券/现金/黄金)]
    
    A --> G[点击"智能再平衡"]
    G --> H[AI 生成调整建议]
    H --> I[方案 A: 卖出部分股票, 买入国债]
    H --> J[方案 B: 买入看跌期权对冲]
    
    I --> K[一键调仓预览]
    K --> L[确认执行]
```
